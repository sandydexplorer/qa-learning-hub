🌐 GitHub Pages Info

To access this app from any device for free:
1. Create a free account at github.com
2. Create a new repository (e.g. qa-learning-hub)
3. Upload all files from this learning-hub folder to the repository
4. Go to Settings → Pages → Deploy from main branch
5. Your app will be live at https://yourusername.github.io/qa-learning-hub/
6. Open this URL on any device — laptop, phone, tablet!
